Troubleshooting Calorie Data

If you're not seeing calorie data for activities, please check:

  1. Your Strava privacy settings (Settings > Privacy Controls > Activity Data)
  2. Individual activity privacy settings
  3. Whether the activity was recorded with a heart rate monitor

Note: The Strava API may not return calorie data in the list endpoint. For detailed activity information, click the "View" link for any activity.
